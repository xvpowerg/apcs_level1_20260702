題目內容

在一維數線上進行一場模擬遊戲。遊戲規則如下：

  • 初始位置為 0。
  • 初始生命值為 k（1 ≤ k ≤ 20）。
  • 當前生命值為 v 時，角色會往右跳 v 格，也就是從位置 p 移動到 p + v。
  • 每次移動後，如果落在 x1 或 x2 的倍數上，會損失生命值：
    • 若落在 x1 的倍數，生命值減少 y1。
    • 若落在 x2 的倍數，生命值減少 y2。
    • 若同時是 x1 和 x2 的倍數，生命值總共減少 y1 + y2。
  • 當生命值小於等於 0 時，遊戲結束，輸出當前所在的位置。

輸入說明

輸入第一行 k 表示初始生命值。

第二行有 x1 與 y1。

第三行有 x2 與 y2。

輸入範圍

  • k：初始生命值，1 ≤ k ≤ 20
  • x1、x2：兩個陷阱的倍數條件，1 ≤ x1, x2 ≤ 10
  • y1、y2：對應陷阱的扣血值，1 ≤ y1, y2 ≤ 20

子題

  • 40%：x1 = 3、x2 = 4、y1 = y2 = k
  • 60%：無額外限制

輸出說明

輸出遊戲結束所在的位置。

2. 一維數線動畫
藍色人是角色；橘色是 x1 陷阱；紫色是 x2 陷阱；紅色是兩個都踩到
k=1, x1=3, y1=1, x2=4, y2=1
角色位置 x1 倍數陷阱 x2 倍數陷阱 同時踩到兩個陷阱
3. 本題三個重點
寫程式時最容易錯的地方
🦘先跳，再檢查。
一開始位置是 0，但不扣血。每次都是跳完落地後，才看是不是陷阱。
❤️生命值 k 有兩個用途。
它既是目前血量，也是這回合要跳的格數。例如 k=7 就跳 7 格。
⚠️兩個 if，不要用 elif。
如果同時是 x1 和 x2 的倍數，要扣 y1+y2，所以兩個陷阱都要檢查。
4. 回合表
每一回合到底怎麼跳、怎麼扣血
回合 跳之前位置 跳之前生命 跳幾格 跳到位置 踩到什麼 扣血後生命
5. 白話記法
學生只要記住這個流程
  1. 站在 pos。一開始 pos = 0。
  2. 看血量 k。血量多少，就往右跳幾格。
  3. 落地後看陷阱。能被 x1 整除，扣 y1；能被 x2 整除，扣 y2。
  4. 如果血量 k ≤ 0。遊戲結束，輸出現在的位置 pos。

6
7while k > 0: # 只要還有生命，就繼續玩
8 pos += k # 生命值是多少，就往右跳幾格
9
10 if pos % x1 == 0: # 如果落在 x1 的倍數
11 k -= y1 # 扣 y1 血

白話重點

  • 一開始在位置 0，但要先移動，才檢查是否踩到陷阱。
  • 每回合先做 pos += k：生命值多少，就往右跳多少格。
  • 落地後分別檢查 x1 與 x2，所以必須使用兩個 if。
  • 當生命值 k ≤ 0，停止跳躍，輸出目前位置。
